一、栈（顺序存储&链式存储）
1.后进先出
2.pop的时候不用删数据，只需对top位进行操作，下次push会覆盖原来的数据
3.例：用一个数组做两个栈，要求空间利用最大化
  两头分别放，设置top1,top2,当top2=top1+1栈满。
  代码实现，用tag（==1||==2）区分两个栈。
4.顺序存储实现&链式存储实现-链栈：插入和删除只能在栈顶top进行
  在这一侧插入/删除才方便  o->o->o->o->o  这一侧插入/删除不知道把top指针指回到哪里
5.链式存储push：申请，加入。

二、栈应用：表达式求值
2+9/3-5      293/+5-
1.运算数相对顺序不变，直接输出
2.运算符号-需要存储等待中的运算符号
          -将当前运算符号与等待中的最后一个运算符号优先级进行比较，如果比他低，那么等待中的那个符号就可以输出
3.左括号压到栈，遇到右括号则把符号一个一个pop直到遇到左括号
4.其他应用：函数调用、深度优先搜索、回溯算法。。。。。。
